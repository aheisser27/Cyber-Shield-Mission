# 🛡️ The Cyber Shield Mission: Web CTF

Welcome to the Cyber Shield Mission! This is a beginner-friendly Capture The Flag (CTF) activity designed to introduce students to web vulnerabilities, source code analysis, and browser diagnostic tools. 

This project bridges the gap between everyday internet use and professional cybersecurity analysis by turning the browser into a "Digital X-Ray."

## 🎯 The Narrative: Hunter vs. Shield
Students act as the "Shield" for their community. Their objective is to track a "Hunter" (a simulated threat actor) who has compromised various systems. The Hunter leaves digital footprints—known as "Flags"—hidden inside the architecture of the websites. Students must use critical thinking and technical translation to uncover them.

## 📂 Mission Levels

* **Level 1: The Apprentice** * **Skill:** HTML Source Code Inspection.
  * **Objective:** Understand that every website has a "backstage" by viewing the page source to find hidden comments.
* **Level 2: The Analyst** * **Skill:** Data Parsing & Search Optimization.
  * **Objective:** Locate a specific anomaly inside a massive server log using `Ctrl+F` to separate the signal from the noise.
* **Level 3: The Specialist** * **Skill:** Cryptography & Data Encoding.
  * **Objective:** Identify scrambled text and use external tools (like CyberChef) to decode a Base64 cipher.
* **Level 4: The Elite** * **Skill:** Browser Developer Tools.
  * **Objective:** Move beyond the HTML structure and investigate the browser's diagnostic console to find scripts executing in the background.

## 🤝 Core Bootcamp Standards
This activity is built on four core standards designed to leverage the cultural wealth of our students:

1. **Translation:** We remove unnecessary jargon. If a technical concept cannot be explained simply, it is not understood well enough.
2. **Resilience:** Technical errors are learning milestones, not dead ends. We stay locked onto the problem until it is solved.
3. **Social and Familial Capital:** Cybersecurity is a team sport. If you finish your mission early, your next objective is to help the person next to you finish theirs. 
4. **Professional Identity:** We hold ourselves to the standard of professional Cyber Defense Analysts, proving that excellence is defined by the quality of our work.
